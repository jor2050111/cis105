# CIS105 Glossary

This glossary is the single source of truth for all technical terms used in the CIS105 textbook. All chapters must use these exact definitions and capitalizations.

**Usage rules:**
* Define every term on first use in each chapter (bold the term)
* Reference this glossary in Key Terms sections (do not duplicate definitions)
* Add new terms alphabetically as chapters are written
* Use exact capitalizations specified here

---

## A

**AI assistant:** Software that responds to plain-language requests: it can draft text, summarize a document, suggest a formula, or answer a question in full sentences. AI assistants are now built into everyday programs such as Word and browsers.

**AI PC:** A personal computer that adds an NPU (neural processing unit) alongside its processor and graphics chip, so AI features can run directly on the device instead of on a distant server.

**app:** The everyday name for an application program, first popularized on phones and tablets and now used for software on every screen.

**application software:** Software that does the user's tasks, such as writing a letter, building a worksheet, or placing an order. Word, Excel, PowerPoint, and browsers are application software.

**artificial intelligence (AI):** The broad family of software techniques that let computers perform tasks that normally require human judgment, such as recognizing a face or drafting a paragraph.

## C

**cloud computing:** Using computing services, such as storage, software, and processing power, delivered over the Internet from remote data centers.

**connectivity:** The communication links that move data between devices, from home Wi-Fi to the Internet itself. One of the six parts of an information system.

## D

**data:** Raw, unorganized facts: numbers, words, images, and sounds. Data becomes information when it is organized into something meaningful.

**database:** An organized collection of related data, built for searching and connecting records. This course builds databases in Airtable.

**desktop:** A personal computer that gives up portability for power, more ports, and easy upgrades and repairs.

**device driver:** A small system program that teaches the operating system to communicate with a specific hardware device, such as a printer, camera, or mouse.

**document:** A file of formatted text, such as a letter, memo, report, or flyer, created with a word processor such as Word.

## E

**Editor:** Word's built-in spelling and grammar checker, found on the Review tab. It flags suspected errors and suggests fixes, but it cannot judge meaning.

**end user:** The person a finished system or application exists to serve, as distinct from the people who build or maintain it.

## F

**file extension:** The short code after the dot in a file name, such as `.docx` or `.xlsx`, that tells the computer (and you) what kind of file it is.

**Find and Replace:** The Word tool that changes every occurrence of a phrase at once. Opened with Ctrl+H on Windows or Edit > Find > Replace on a Mac.

## G

**generative AI:** The branch of artificial intelligence that creates new content, including text, images, and audio, in response to prompts.

## H

**hardware:** The physical equipment in an information system: phones, laptops, servers, screens, and payment readers. One of the six parts of an information system.

## I

**information:** Data that has been organized or processed into something meaningful enough to support a decision.

**information system:** The combination of six parts working together to turn data into useful work: people, procedures, software, hardware, data, and connectivity.

**Internet of Things (IoT):** The growing collection of everyday objects, such as doorbells, thermostats, and delivery trucks, fitted with sensors and network connections so they can send data over the Internet.

## L

**laptop:** A portable personal computer that folds a full keyboard, screen, and battery into one case. The standard machine for producing documents, worksheets, and presentations.

## N

**NPU (neural processing unit):** A chip built specifically for running AI features directly on a device. The defining component of an AI PC.

## O

**operating system:** The central piece of system software: the program that starts the machine, manages its hardware, and launches everything else. Windows, macOS, iOS, and Android are operating systems. OS after first mention.

## P

**PDF:** A file format that locks a document's layout so it looks the same on every device. The standard format for delivering finished documents.

**presentation:** A file of slides that support a talk, created with presentation software such as PowerPoint.

**procedures:** The rules and steps people follow when they work with a system. One of the six parts of an information system.

**prompt:** The plain-language request you type or speak to an AI assistant.

## R

**ribbon:** The strip of tabs and buttons across the top of the Word window (and other Microsoft 365 apps) that organizes the program's tools by task.

## S

**server:** A computer that provides data or services to other computers over a network. Servers run around the clock in data centers.

**smartphone:** A pocket-sized computer with phone service. The most widely owned computer in the world.

**software:** The instructions that tell a computer what to do. Programs. One of the six parts of an information system.

**system software:** Software that runs the computer itself instead of doing user tasks. The operating system, utilities, and device drivers are system software.

## T

**tablet:** A personal computer built around a large touchscreen, lighter than a laptop and larger than a phone.

## U

**utility:** A system program that does computer housekeeping, such as backing up files, cleaning storage, or guarding against malware.

## W

**wearable:** A small computer worn on the body, such as a smartwatch or fitness band, that gathers data throughout the day.

**worksheet:** A grid of rows and columns holding numbers, formulas, and charts, created with spreadsheet software such as Excel.
