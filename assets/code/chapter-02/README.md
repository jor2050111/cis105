## Data dictionary

**ai-briefing.docx:** The briefing Maya's AI assistant produced about the web and online bean sales: the prompt she used, a confident introduction, eight numbered claims, and a confident closing. Some claims are correct, some are not, and the tone never changes. Which are which is the lab's work. Do not mark up this file. Record your verdicts in your own memo.
