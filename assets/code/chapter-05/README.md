## Data dictionary

**spec-sheets.docx:** Cactus Wren Repair's counter reference: full spec tables for three refurbished devices (Desert Falcon 15 laptop, Stone Tower desktop, Bloom Pad 11 tablet), three customer profiles, and Amir Haddad's shop notes. All values in this file are correct. Students verify slides against it and never edit it.

**spec-outline.docx / spec-outline.rtf:** The same shop content as an importable outline: 9 slide titles styled Heading 1 and 36 bullets styled Heading 2, identical content in both files. The outline ships with three deliberate flaws the lab asks students to find and repair: one Desert Falcon bullet mislabels 512 GB of storage as "memory," one bullet (the BIOS version) is useless to walk-in customers and belongs in the notes, and the Side by Side slide is a wall of numbers awaiting a table or chart. Do not edit these files. Import them and make every fix in the deck.

## Cast notes

Cactus Wren Repair is the mobile device repair shop in the course's recurring cast, owned by Amir Haddad, two blocks from Saguaro Hall (Chapter 4). Its device inventory returns in Chapter 6's Excel workbook.
