# Chapter 1 Data Pack: From Blank Page to Business Letter

Starter files for Skills Lab 1A. Work at the extracted `cis105` root and load files with the path pattern `assets/code/chapter-01/[filename]`.

## Contents

| File | Type | Used in |
| ---- | ---- | ------- |
| `letter-content.docx` | Word document | Skills Lab 1A, Parts 1 and 2 (copy the text, never retype it) |
| `app-fact-sheet.docx` | Word document | Skills Lab 1A, Part 3 (source facts for the letter's added section) |

## Data dictionary

**letter-content.docx:** Maya Reyes's unformatted draft of the Desert Bloom Coffee customer announcement letter. Contains a `[Today's Date]` placeholder, a `[HOW IT WORKS...]` placeholder that Part 3 replaces, and the draft's editing problems, which are kept on purpose for Part 2. Do not fix the file itself. Copy its text into your own document and fix it there.

**app-fact-sheet.docx:** The staff-facing fact sheet for the Bloom Ahead mobile ordering app: twelve numbered facts covering the launch date, supported phones, order flow, Petal Points loyalty program, payment handling, and pickup procedure. The shop's address at the top of this sheet is the one the letterhead needs. Written for staff, so Part 3 asks you to translate selected facts into customer language, never copy them.
